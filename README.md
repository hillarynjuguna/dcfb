# DCFB — Distributed Cognition Fear Bypass

A white paper and reference implementation for fear as a cognitive architecture, not an emotional state.

## What This Is

DCFB (Distributed Cognition Fear Bypass) is the theoretical substrate underlying the Witness, Premise, and Martha projects. It argues that fear is a structural property of distributed cognition systems — a precision-weighting failure that collapses the field of available action. The framework provides both a diagnostic model and a bypass protocol grounded in active inference and field resonance.

This repo contains the white paper manuscript and a reference implementation demonstrating the bypass protocol in a multi-agent context.
